SunHouse is a clean and modern Real Estate HTML Template.

File structure

  • assets/
    • css/
      • components.css (Style for global components)
      • layout.css  (Main style for template)
      • responsive.css (Style for mobile device view)
    • font/
    • images/
    • js/
      • pages/...  (Script for each page)
      • main.js (Script for global components)
    • libs/  (included global plugins library)
  • HTML files
